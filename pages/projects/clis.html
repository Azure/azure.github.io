Azure CLI 2.0

Azure CLI 2.0 is your command line for Microsoft Azure that you can use from anywhere.

With Azure CLI 2.0, you can create, manage, and delete services on the command line via cmd.exe, bash or {your shell} on the operating system of your choice.

Documentation
/azure-cli
Install
OSX and Linux
curl -L https://aka.ms/InstallAzureCli | bash
Windows
pip install --user azure-cli
